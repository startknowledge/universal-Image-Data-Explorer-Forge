## ✨ Features

### 🖼️ Image Input Support
- PNG · JPEG · WEBP · BMP · GIF
- Drag & drop or click to upload

### 🔄 60+ Conversion Formats (8 Categories)

#### A. Binary / Low Level
- Binary (bit string)
- Hexadecimal
- Octal
- Byte Array
- Uint8Array
- Int16Array / Int32Array
- Float32Array / Float64Array
- Bit Stream

#### B. Encoding
- Base64 · Base32 · Base85
- Data URL
- ASCII / UTF-8 Text
- JSON / XML

#### C. Pixel Data
- RGB Matrix · RGBA Matrix
- Grayscale Matrix
- HSL / HSV · CMYK · YUV · LAB
- Pixel Coordinates
- Histogram Data

#### D. Programming Formats
- C Array (`unsigned char[]`)
- C++ Array
- Python Bytes (`bytes([...])`)
- Java Byte Array (`byte[]`)
- Go Slice
- Rust Array
- JavaScript Buffer
- PHP Binary (`pack()`)

#### E. Visualization
- ASCII Art · Emoji Art
- Pixel Grid
- Heatmap (2D intensity)

#### F. Compression
- Run Length Encoding (RLE)
- Huffman (symbol table)
- LZW stub
- DCT block preview

#### G. Security / Hashing
- MD5 · SHA1 · SHA256
- CRC32
- File Signature (magic bytes)

#### H. AI / Data Science
- NumPy Array (`np.array([...])`)
- TensorFlow Format
- CSV Pixel Data
- Feature Vector

### 🔁 Reverse Reconstruction
- **Binary → Image** (paste bits + set dimensions)
- **Hex → Image** (coming soon)
- **Base64 → Image** (automatic)

### 🛠️ Advanced Viewers
- **Binary Viewer** – bit-level with toggle
- **Hex Editor** – editable, cursor support
- **Pixel Inspector** – click on preview to see RGB/Hex/Binary
- **Metadata Viewer** – dimensions, file size, type, EXIF
- **Color Analyzer** – dominant colors + histogram

### 📦 Download System
- Export in **all supported formats**
- File extensions: `.txt`, `.c`, `.cpp`, `.py`, `.java`, `.js`, `.bin`, `.csv`, `.json`, `.xml`, `.h`, `.rs`, `.go`, `.php`
- **Email gate** – one-time email registration (stored locally)

- Affiliate network stubs (Amazon, CJ, ShareASale, Impact, Rakuten, Awin, FlexOffers, Viglink)
